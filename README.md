# Movie Search Application

## Overview

The Movie Tracking Application is a Salesforce-based project developed using Apex and Lightning Web Components (LWC) with external API integration capabilities. The application retrieves and displays movie-related information dynamically using asynchronous Apex callouts and reusable Lightning components.

This project focused on API integration, external data handling, asynchronous operations, and dynamic UI rendering within Salesforce.

---

## Features

* External movie API integration
* Asynchronous Apex callouts
* Dynamic movie data rendering
* Reusable Lightning Web Components
* API response handling
* Interactive UI design
* External data management
